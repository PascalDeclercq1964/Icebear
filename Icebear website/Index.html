Icebear reporting for .net

Icebear reporting for .net is a reporting solution that allows to print reports or generate them as PDF. 
The reports are defined in code, there is no designer. 
No designer?? Well no, but give it a try! Be you will make your first report faster than with any other tool with designer.

Download Learn more

Getting started

This is an example of a very simple report: 

Report report = new Report();
report.DataSource = customers.OrderBy(c => c.CompanyName);
report.AddField("Id", 50, HeaderLabel:"ID");
report.AddField("CompanyName", 100, HeaderLabel:"Name");
report.AddField("Address", 100, HeaderLabel:"Address");
report.AddField("City", 100, HeaderLabel:"City");

report.GeneratePDF($"c:\\temp\\Report {DateTime.Now:yyyyMMddhhmmss}.pdf");

Not the fanciest report ever made, but it's a start!
