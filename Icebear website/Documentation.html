The simplest report

To create a very basic report use the following code: 
Report report = new Report();
report.DataSource = myData;	// myData is an IEnumerable of a strongly typed object as datasource
Fields are added starting at the left margin. Optionally a headerlabel can be specified 
report.AddField("FieldName", width);  
report.AddField("FieldName", width, HeaderLabel:"Field Label");
Generate the report to a PDF or print it to a printer of your choice: 
          
report.GeneratePDF($"c:\\temp\\Report{DateTime.Now:yyyyMMddhhmmss}.pdf"); 
report.Print(WindowsPrinterName);
